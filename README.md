# nfkutvikleroppgave

# What does it do?
Collects information on a list of organization numbers and inserts them into a database structure and displays them in a web interface.

# Built With
##  Backend
Nodejs, express
## Frontend
HTML, JavaScript, Bootstrap5/CSS
## Database
MySQL

- mysql
- express

- xlsx
- dataTables
- Bootstrap5
